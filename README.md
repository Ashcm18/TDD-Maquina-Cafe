# TDD Maquina Cafe

## Descripción
Este proyecto implementa una máquina dispensadora de café utilizando la metodología de Desarrollo Guiado por Pruebas (TDD).

## Funcionalidades
- Selección de tamaño de vaso:
  - Pequeño: 3 oz
  - Mediano: 5 oz
  - Grande: 7 oz
- Selección de cantidad de azúcar.
- Validación de tamaños de vaso.
- Pruebas automatizadas utilizando Python y unittest.

## Tecnologías utilizadas
- Python
- unittest
- Git
- GitHub
- GitHub Actions

## Pipeline CI/CD
El proyecto cuenta con un pipeline de Integración Continua configurado con GitHub Actions, que ejecuta automáticamente las pruebas cada vez que se realiza un push o un pull request.
